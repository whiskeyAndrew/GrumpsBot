# Grumps Twitch Bot

Grumps is a bot for Twitch streams

- Made for Java 17 (but must compile on 9+ versions);
- Made with Spring and Twitch4J frameworks;
- Handles chat messages;
- Stores some data (like IQ counter and etc.) in PostgreSQL DB.
